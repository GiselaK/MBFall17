How to push to Github
Start a Github Project: Have one person in your group do the following steps (If you're unsure if you did a step right or want to learn about what each step does, grab a volunteer/instructor!):
  1. Create a folder on your Desktop
    You can do this by typing (without the parenthesis):
    cd ~/Desktop
    mkdir (folder-name)
  2. Switch folder name with the one you just created and type cd ~/Desktop/(folder-name) in the terminal (without the parenthesis)
  3. Create repo on Github
  4. Copy the code on Github under "â€¦or create a new repository on the command line" and paste it into the terminal
  5. Create an index.html file inside the folder that says "Hello World"
  6. Paste git checkout -b gh-pages
  7. Continue to Update Project On Github steps
  8. (github-username).github.io/(repository-name) Should say "Hello World" (Grab a volunteer/instructor if it doesn't!)
  9. If you're working with others, go to settings then collaborators and add your team members usernames
Have all other team members do the below (Strongly advice team members to NOT work on the same files! One should push changes to one file before another team member touches that file):
  1. Navigate to your repository
  2. Hit the green clone button & copy the url
  3. Type (without the parenthesis) git clone (paste url) in the terminal
Update Project On Github: Type this in the terminal (without the parenthesis):
git add .
git commit -m "(write changes you've made)"
git push origin gh-pages
