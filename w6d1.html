    1. Make a function which takes an array and index as parameters. Then, returns the item at the index passed in.
      Examples:
      getItem([1,2,3], 0); -> 1
      getItem([1,2,3], 1); -> 2
      getItem([1,2,3], 2); -> 3
      getItem(["A", "B", "C"], 1); -> "B"

    2. Make a function which logs every element in a array
      Examples:
      log([1,2,3]) -> "1 2 3";
      log(["hi","hello","hey"]); -> "hi hello hey"
      Improve this function by making it print every other element Examples:
      log([1,2,3]) -> "1 3";
      log(["hi","hello","hey"]); -> "hi hey"

    3. Make a function which takes an array of students scores and returns the highest score
      Example:
      highestScore([79,86,83]) -> 86
      highestScore([17,44,29,46]) -> 46

      Example:
      reverseArr(["a","b","c"]) -> ["c", "b", "a"];
      reverseArr(["evening", "afternoon", "morning"]) -> ["morning", "afternoon", "evening"];
